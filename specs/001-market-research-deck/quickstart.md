# Quickstart: AI Dating App Market Research Deck

## Run the deck

```bash
cd react-vite
npm install
npm run dev
```

## What to edit

- **Deck UI**: `react-vite/src/App.jsx` (will move into `react-vite/src/deck/*` per plan)
- **Global styles**: `react-vite/src/index.css` (Tailwind entrypoint)
- **Content model**: `specs/001-market-research-deck/data-model.md`
- **Research + sources**: `specs/001-market-research-deck/research.md`

## Quality gates (must pass before sharing)

- Every number/comparison has a source + date (sources appendix)
- Mobile readable (≥16px body), touch targets ≥44×44px
- WCAG AA contrast + keyboard focus visible
- Microanimations respect reduced motion
- No heavy assets that slow load on mobile
