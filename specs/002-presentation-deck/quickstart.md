# Quickstart (Phase 1): Run the deck locally

## Prerequisites

- Node.js installed

## Run the deck

From repository root:

```bash
cd react-vite
npm run dev
```

Then open the URL printed in the terminal.

## Build a static bundle

```bash
cd react-vite
npm run build
npm run preview
```

## Update slide content

Edit:

- `react-vite/src/content/deck.content.js`

## Update photos

Photos are served from:

- `react-vite/public/photos/first-prompt/`
